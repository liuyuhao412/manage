作品集

Web客户端人脸登录

项目时间：2023年8月 - 2023年9月

该系统实现了基于人脸识别的登录功能。后端使用 Flask 框架建立了数据库和 API 接口，前端采用 HTML5、CSS 和 JavaScript 进行页面布局。系统利用人脸识别算法进行用户登录验证。

主要包括：

  • 前端设计了登录页面，使用浏览器自带的摄像头接口来捕获用户的实时视频。
  • 通过 Canvas 对视频帧进行处理，捕获人脸图像，并将其转换为 base64 数据格式传输到后端。
  • 后端建立了数据库和 API 接口，并实现了人脸识别算法用于用户登录验证。

通过该项目，我掌握了前后端人脸识别技术的实现，并提高了对 Flask 框架的使用和 Web 开发技能的理解。

基于大模型的知识问答系统

项目时间：2024年1月 - 2024年3月

该系统为毕业设计项目，基于前后端分离架构开发，旨在实现基于大模型的知识问答功能。前端使用了 Vue 3 和 TypeScript 技术栈，后端采用 Flask 框架，数据存储使用 MySQL 数据库。在 GPU 服务器上部署了 LangChain-ChatChat 模型，通过 API 调用完成问答系统的文件上传及问答功能。

主要包括：

  • 实现了用户管理等功能
  • 管理员可以上传文件，来管理知识问答库和问答参数的设置
  • 用户可以对基于问答库的问答

通过该项目，我提升了对前后端分离开发的理解，解决了实际开发中的跨域问题，并掌握了部署大模型及其 API 调用的实际技能。
